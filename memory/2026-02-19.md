## WebView Pattern (Current Standard)
All tiger apps use same logic:
1. App launch → HTTP request to `domain.org/click.php` with RedirectTracker
2. Follows all redirects, records final URL
3. Final URL contains "sites.google.com" → show app, else → WebView
4. Timeout 5 sec → default to showing app
5. Settings/Profile tab → direct WebView load (no check)

Pattern verified in: Golden Path, Tigerroar, Tigerbeing.
