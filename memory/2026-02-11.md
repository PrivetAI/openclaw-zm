## Simulator UI Testing — SOLVED
- Built `/tmp/sim_tap` tool (Swift) that taps iOS Simulator at phone coordinates
- Key insight: must use AX API to find the phone screen GROUP inside Simulator window (offset ≠ window origin)
- Phone screen group found via AXGroup child of Simulator window → get position + size → scale to phone logical coords (393x852)
- Tab bar coords (for custom tab bars): y≈800-815, x divided by number of tabs
- `/tmp/sim_tap <phone_x> <phone_y>` — compiled Swift binary, requires Accessibility permission for node
- node added to Accessibility in System Settings
