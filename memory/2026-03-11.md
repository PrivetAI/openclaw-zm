## Batch 7: Ice Fishing Apps — Tab Icons & App Icons Fixed

### Tab Bar Icons on iOS 26 (CRITICAL LESSON)
- iOS 26 "liquid glass" tab bar does NOT render:
  - SwiftUI Shape `.stroke()` in tabItem
  - `Image("named_asset")` from Assets.xcassets in tabItem
  - Canvas-based Views in tabItem
- **ONLY working approach**: `Image(uiImage:)` with `UIGraphicsImageRenderer` + `.withRenderingMode(.alwaysTemplate)`
- Must use **filled** paths (not stroked, no even-odd holes from double addEllipse)
- Extension pattern that works:
```swift
extension Shape {
    func asTemplateIcon(size: CGFloat = 24) -> UIImage {
        let renderer = UIGraphicsImageRenderer(size: CGSize(width: size, height: size))
        let img = renderer.image { ctx in
            let rect = CGRect(origin: .zero, size: CGSize(width: size, height: size))
            ctx.cgContext.addPath(self.path(in: rect).cgPath)
            ctx.cgContext.setFillColor(UIColor.white.cgColor)
            ctx.cgContext.fillPath()
        }
        return img.withRenderingMode(.alwaysTemplate)
    }
}
```
