### iOS Simulator Automation
Use ios-simulator skill for:
```bash
# Setup
node scripts/ios-sim.mjs select --name "iPhone" --runtime "iOS" --boot

# Install & launch
node scripts/ios-sim.mjs app install --app path/to/App.app
node scripts/ios-sim.mjs app launch --bundle-id com.example.App

# UI automation
node scripts/ios-sim.mjs ui summary --limit 20
node scripts/ios-sim.mjs ui tap --query "Tab Name"
node scripts/ios-sim.mjs ui type --text "input"
node scripts/ios-sim.mjs screenshot --out screenshots/01.png
```
