## WebView RedirectTracker Pattern (Finalized)

**Standard implementation for all apps:**

1. **App.swift**: Launch check with RedirectTracker (URLSessionTaskDelegate)
   - Follows redirects, records final URL
   - If final URL contains "sites.google.com" → show app
   - If final URL is anything else → show WebView fullscreen
   - Timeout 5 seconds → fallback to app

2. **Components needed:**
   - WebPanel (UIViewRepresentable with WKWebView)
   - LoadingScreen (splash with animation)
   - Settings WebView entry (direct load, no check)

3. **Info.plist:** NSAllowsArbitraryLoads = true (required)

4. **Code uniqueness:** Every app has unique variable/struct/class names
   - Examples: BNationRedirectWatcher, AquaRedirectFollower, StarzPathResolver
